# zfs
Fills free disk space with zeroes.

**Requirements:**
--
1. Lua (tested with v5.2);
2. Linux (tested on Lubuntu v17.04).

**Usage:**
--
1. Set "partition" variable.
2. If "zfs_fill.dat" exists in current directory, adjust "f" variable (file will be cleaned).
3. Run script.
